Resource Summary

The Safe Exam Browser can be used to ensure students can only access certain resources when completing summative assessment activities.

Important: Please take great care to ensure the application of these tools does not impact students who require additional supports (for example, assistive technology).

This guide outlines several prebuilt configuration files as well as the basic process for editing or creating your own .seb configuration file for classroom use.

Available SEB Files

A few different configurations were created to address common assessment use cases, including EQAO assessments, math course assessments, and Brightspace / HUB assessments.

Note: SEB is force-installed on student devices. Teachers may need to install it first from Company Portal or Software Centre.

Locating the SEB Configuration Tool

To modify one of the configuration files, first locate the SEB Config Tool on a Windows device:

Windows (C:) / Program Files / SafeExamBrowser / Configuration / SEBConfigTool

Changing a Configuration File

  1. Open the Config File tab in the SEB Config Tool.
  2. Download one of the .seb files and save it locally.
  3. Select Open Settings… to open the file for editing.
  4. Enter the administrator password if prompted.
Shared password: REDACTED

Within the General tab, you can protect the configuration file with an administrator password and optionally create a separate quit / unlock password. This is also where the Start Page URL is configured.

The prebuilt files already point to HTML pages with the required links included, but you can change the start page to any URL that fits your assessment workflow.

Network and Navigation Rules

  • Simply adding a URL is not enough—students also need a way to navigate to that link.
  • If you create your own start page, it must contain links to the permitted resources.
  • Students do not have access to the browser address bar, so they cannot manually type URLs.

If you need access to an entire site instead of one page, you can use a wildcard after the root URL.

hwdsb.elearningontario.ca/*

After making your changes, return to the Config File tab and choose Save Settings As… to create a new configuration file.

Using OneNote as the Start Page

The OneNote option can support either a dedicated assessment notebook or an open-book approach where students can access their class notes during an assessment.

  1. Open the notebook in OneNote Online.
  2. Select the Class Notebook tab.
  3. Choose Manage.
  4. Copy the notebook link shown in the bottom-right corner of the management page.
Important: If the copied link begins with onenote:, remove that prefix so the URL begins with https:// before placing it in the Start Page field.

The OneNote template files already include the necessary login and redirect URLs under the Network tab; teachers only need to insert the notebook URL in the Start Page field.

Preventing Students from Editing OneNote Assessments

The easy way: Page Locking

  1. Open the Class Notebook tab.
  2. Select Review Student Work.
  3. Choose the section where the page was distributed.
  4. Select the page you want to review.
  5. Use Page Locking to lock the page.

The heavy-handed option: Remove notebook access

  1. Open Microsoft Teams.
  2. Go to the team’s General channel.
  3. Open Files, then open the folder in SharePoint.
  4. Open Site contents.
  5. Open the Site Assets folder and locate the class notebook.
  6. Use Manage Access, then Advanced settings.
  7. Select student accounts and remove user permissions.
Caution: Make sure you do not remove your own permissions.

Repairing OneNote Permissions

When students need to be added back after permissions were removed, use the public OneNote Class Notebook permission repair tool:

https://www.onenote.com/classnotebook/notebook-fix?auth=2

Locate the notebook and choose Fix Permissions. This will restore student access based on membership in the underlying Team.
